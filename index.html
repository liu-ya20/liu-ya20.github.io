    Datasets used in Music Tagging

    GTZAN Dataset

    • GTZAN Dataset: The GTZAN dataset is the most-used public dataset for evaluation in machine listening research for music genre recognition (MGR). The files were collected in 2000-2001 from a variety of sources including personal CDs, radio, microphone recordings, in order to represent a variety of recording conditions

    • 3.71GiB

    • content : 10 genres, each represented by 100 tracks. The tracks are all 22050Hz Mono 16-bit audio files in .wav format.

    Million Song Dataset

    • The core of the dataset is the feature analysis and metadata for one million songs, provided by The Echo Nest. The dataset does not include any audio, only the derived features. Note, however, that sample audio can be fetched from services like 7digital, using code we provide.The Million Song Dataset is also a cluster of complementary datasets contributed by the community:

    • 280GB（To start your own experiments, you can download the entire dataset (280 GB). We also provide a subset of 10,000 songs (1%, 1.8 GB compressed) for a quick taste.）

    Ballroom

    • 该数据集由来自舞厅的音频数据组成。以实际音频格式提供了许多舞蹈风格的一些音频片段。
    • 14GB（compressed）
    • content: 698 * approx30s

    Free Music Archive(FMA)

    • 数据集由音频（full-length and HQ），预先计算的特征（ pre-computed features）以及音轨和用户级元数据（metadata）组成。

    • fma_small.zip: 8,000 tracks of 30s, 8 balanced genres (GTZAN-like) (7.2 GiB)
      fma_medium.zip: 25,000 tracks of 30s, 16 unbalanced genres (22 GiB)
      fma_large.zip: 106,574 tracks of 30s, 161 unbalanced genres (93 GiB)
      fma_full.zip: 106,574 untrimmed tracks, 161 unbalanced genres (879 GiB)

    • content:
      tracks.csv: per track metadata such as ID, title, artist, genres, tags and play counts, for all 106,574 tracks.
      genres.csv: all 163 genres with name and parent (used to infer the genre hierarchy and top-level genres).
      features.csv: common features extracted with librosa.
      echonest.csv: audio features provided by Echonest (now Spotify) for a subset of 13,129 tracks.

    MagnaTagATune

    • MagnaTagATune dataset contains 25,863 music clips. Each clip is a 29-seconds-long excerpt belonging to one of the 5223 songs, 445 albums and 230 artists. The clips span a broad range of genres like Classical, New Age, Electronica, Rock, Pop, World, Jazz, Blues, Metal, Punk, and more. Each audio clip is supplied with a vector of binary annotations of 188 tags. These annotations are obtained by humans playing the two-player online TagATune game. In this game, the two players are either presented with the same or a different audio clip. Subsequently, they are asked to come up with tags for their specific audio clip. Afterward, players view each other’s tags and are asked to decide whether they were presented the same audio clip. Tags are only assigned when more than two players agreed. The annotations include tags like ’singer’, ’no singer’, ’violin’, ’drums’, ’classical’, ’jazz’. The top 50 most popular tags are typically used for evaluation to ensure that there is enough training data for each tag. There are 16 parts, and researchers comonnly use parts 1-12 for training, part 13 for validation and parts 14-16 for testing.

    • contain: Clip metadata in CSV format
      Clip metadata as MYSQL database dump
      Audio features extracted by The Echo Nest API 1.0
      Tag annotations collected from game players in CSV format
      Similarity data in CSV format
      Audio data: mp3 32bps file1(1GB) file2(1GB) file3(1GB)

    DAY 3

    Music Generation DataSet

    Lakh

    • The Lakh MIDI dataset is a collection of 176,581 unique MIDI files, 45,129 of which have been matched and aligned to entries in the Million Song Dataset. Its goal is to facilitate large-scale music information retrieval, both symbolic (using the MIDI files alone) and audio content-based (using information extracted from the MIDI files as annotations for the matched audio files).

    • about 4GB(conpressed)

    Nottingham Database

    • ABC-version
    • midi-version

    This is a collection of 1200 British and American folk tunes, (hornpipe, jigs, and etc.) that was created by Eric Foxley and posted on Eric Foxley’s Music Database. The database was converted to abc music notation format and was posted on abc.sourceforge.net.

    ADL PianoMiDI

    • The ADL Piano MIDI is a dataset of 11,086 piano pieces from different genres. This dataset is based on the Lakh MIDI dataset, which is a collection on 45,129 unique MIDI files that have been matched to entries in the Million Song Dataset.

    GiantMIDI-Piano ( Bytedance )

    • classical piano MIDI dataset contains 10,855 MIDI files of 2,786 composers. The curated subset by constraining composer surnames contains 7,236 MIDI files of 1,787 composers. GiantMIDI-Piano are transcribed from live recordings with a high-resolution piano transcription system

    一个人活着的意义是什么？

    我给自己提出这个问题，稍作思考后我给出这样的正经回答：一个人活着的意义在于与世界产生的种种联系。我作为一个个体，能够参与到社会演进的过程中，世界因为我的存在而不同，这就是我活着的意义。
    但有意义只是一个基础的属性。就好像沙滩上满是贝壳，也有珍珠，贝壳有意义，但路过的人并不都会去拣。
    现实社会里，人并不都会将自己活着的意义像如上回答一样定义得理想而无害，包括我自己。在懵懂的幼年，穿着开裆裤玩泥巴的年纪，我忙碌于迎接世界的新奇、善意与挑战，我不会思考自己存在的意义，我只会给出面向外界的判断：这个树上的果子很香，我喜欢这个果子的气味；和小A玩的很开心，我喜欢和小A见面。

    当开始进入幼儿园，和更多的小朋友坐在一起，我开始有了辨识自我的意识。我发现，我喜欢吃黄瓜，但小B不喜欢；我能在滑梯上上蹿下跳，但小C不行。基于个人面对世界的好恶态度，我建立起最初的第一维的个人价值。同时，当某次活动我得到了sticker，某一个科目里我开始展现实力，某一个老师笑呵呵的对我说你真棒，我开始觉得自己与众不同，也就开始建立起对第二维自我价值的初步认知：我在一些事情上做的好、做的出众，我很棒。但就算是我没有做好一件事，或是被批评，我也不会在意，只要哭一鼻子就会忘记所有烦恼，我相信这是所有孩子都有的魔法。

    进入小学，也正式进入学生时代，学业变成了一条通用的赛道。周期更新的排名进一步巩固了我对于自我价值的第二维认知，并且在上面添加了一点现实的、稍显无情的“比较”。我取得了靠前的名次，说明我在学习上的努力或天赋是出众的，我很棒。倘若有个邻居阿姨做客说出了“你家孩子真优秀，再看看我家的，不好好学习！”这样的话，我会害羞着偷偷开心，默认地接受了这样一份基于成绩，却针对完整个体的认可。那时候的我，并不会思考区分“你家孩子真优秀”和“你家孩子成绩真好”之间的区别。

    总体上讲，这个阶段我很少会自发地去确认自己的价值所在。毕竟处于没有压力、充满乐趣的年纪，和同桌只需要一根钢尺和一块橡皮，就足够打发一下午的课间时光。在这样无忧无虑的年纪，我交到了朋友，也开始懂事，感受到朋友和父母的关爱的珍贵与意义，我也逐步建立起第四维的个人价值，可以陈述为：我能活在这样一个世界，有朋友小C和爸爸妈妈的陪伴，真是幸福的事情。感受到幸福，让我确认到自己的价值。现在回想，这就相当于把与世界上一部分人划进了自己生命共同体的一部分，我希望看到他们快乐，就像希望自己快乐一样。

    （插一嘴） 回忆自己的小学时光，好像想起了一些熟悉的自己。

    初高中时代，不仅更加关注学习成绩，也更加在意自己的外貌，希望自己得到更多人的认可，能够更受女孩子欢迎。我的个人价值里学习成绩的第二维占比越来越重，同时也添加了第三维个人价值：外在。我认识到外在看起来受欢迎，这也会是我存在的意义之一。这一维的个人价值从小学就在逐步累积，但进入青春期才真正开始蓬勃发展。看到外在美的男生、女生，我感到羡慕、喜欢，推己及人，我也希望自己能够像他们一样被大家喜欢。从这个角度来看，这是一维由荷尔蒙参与的个人价值维度。

    紧张的高中学习生活了越来越频繁的考试不断加深第二维个人价值的比重。同时，我也在课业之外的生活中找寻第一维个人价值：我听了adele的歌，成为了喜欢adele的一小部分人；我上了课外英语兴趣班，额外学习了与课内大家学习的不一样的新的知识；我学习了钢琴，让我成为了会弹琴的小部分人。

    就是这样建立起的自我价值体系，在高考之前能够完美地维持，进入大学后就面临崩溃的边缘。

    当成绩突然从引以为傲的资本变成羞于启齿的事情，我的第二维个人价值在大学里就再也没有存在的余地。我清楚地认识到，和一些同学相比，我的知识基础是存在差距的，但这不是我愿意承认的事情，尤其是这种针对“知识基础”的差距逐渐产生对“个人能力”的怀疑的时候。我更愿意承认的是我没有认真投入地学。我想，这或许是导致我形成现在喜欢拖延，缺乏目标和规划的原因之一。这种“我没有投入全部”的心理暗示鼓励我在一些学习之外的活动上投入更多精力，进而能够为自己成绩上的失利找到一个“体面的开脱”。

    我姑且将这种心理和表现定义为第二维个人价值的转化：一部分转化到新的第五维个人价值：个人能力 之中，一部分根据我时间的安排转化到第一和第三维个人价值里。我烫了一个发型，我认为自己增加了自己外在的价值；我去琴房练琴、去唱歌、去看动漫，我认为自己增加了第一维度的价值。但在这个建立于比较的体系里，在任何一个方向上，只要有人远强于我，那一维度的价值占比的提升就存在一个瓶颈。看到键盘队同学们的演奏，看到校园里帅气美丽的男生女生，我的第一、第三维度个人价值就注定没有太多的提升空间。

    那么第四维呢？在大学完全陌生的环境里，只有屈指可数的几位高中同学能够与我一起分享我的生活，第四维个人价值缺乏联系，我也常常感到孤独和无人诉说。

    这就是价值体系面临崩溃的原因。说了这么多，总结起来就是，在基于“比较”建立、且基于“比较”进行调整的价值判断准则下，每一方面都有一些人强过我，甚至有人在主要的方向处处强过我，就会开始怀疑自己的个人价值。
